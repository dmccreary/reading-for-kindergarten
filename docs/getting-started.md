# Getting Started

Welcome to Reading for Kindergarten! Here's how to get started with this textbook.

## How to Use This Textbook

1. **Navigate the chapters** - Use the left sidebar to move through chapters in order
2. **Try the MicroSims** - Interactive simulations help reinforce concepts
3. **Take the quizzes** - Test your understanding after each chapter
4. **Review the glossary** - Look up unfamiliar terms

## For Parents and Teachers

This textbook is designed to be used with adult guidance. Each chapter includes activities that can be done together.
